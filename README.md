### Coin-Keeper!

This is a simple app that allows you to keep track of your expenses. It is built with React-Native and Expo. 

## Installation

1. Clone the repo
```sh
git clone https://github.com/gomeznahuel/Coin-Keeper.git
```

2. Install NPM packages
```sh
npm install
```

3. Run the app
```sh
expo start
```
